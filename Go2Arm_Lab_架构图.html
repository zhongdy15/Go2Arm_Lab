② 训练时数据流（从命令行到 checkpoint）

   命令行: python train.py --task Isaac-Go2Arm-Flat --headless
                │
                ▼
   config/__init__.py          ← gym.register("Isaac-Go2Arm-Flat", kwargs={
                │                     "env_cfg_entry_point": "flat_env_cfg:Go2ARMFlatEnvCfg",
                │ 注册表查找             "rsl_rl_cfg_entry_point": "rsl_rl_ppo_cfg:Go2ArmFlatPPORunnerCfg"
                ▼                 })
   @hydra_task_config           ← 动态 import 并实例化两个 cfg 对象
    ├── flat_env_cfg.py         ← Go2ARMFlatEnvCfg（覆盖奖励/课程参数）
    │       └── go2arm_lab_env_cfg.py   ← 完整 MDP 结构（继承基类）
    │               └── go2arm_articulation_cfg.py  ← 加载 go2_arm.usd
    └── rsl_rl_ppo_cfg.py       ← PPO 超参数 + 网络架构配置
                │
                ▼
   gym.make()                   ← 实例化 ManagerRLEnv
    ├── 加载地形 USD
    ├── 生成 N 个并行机器人
    └── 初始化所有 Manager（Obs / Reward / Event / Action / Curriculum）
                │
                ▼
   VecEnvWrapper.py             ← 将 obs 字典拆分为三部分:
                │                   prop_obs  [N, 52]      本体感知（当前帧）
                │                   priv_obs  [N, 27]      特权观测（训练专用）
                │                   hist_obs  [N, 10×52]   10步历史（供编码器）
                ▼
   OnPolicyRunner               ← 训练主循环（每次迭代）:
    │  ┌─────────────────────────────────────────────────────┐
    │  │ Step 1: 采集 24 步 rollout → rollout_storage        │
    │  │ Step 2: 计算 GAE 广义优势估计                        │
    │  │ Step 3: 5 轮 × 4 mini-batch → PPO 梯度更新          │
    │  │          ├─ actor_critic.py  前向推理                │
    │  │          └─ ppo.py          裁剪目标函数 + 价值损失  │
    │  │ Step 4: 混合调度 mixing_schedule 逐渐切换到历史编码  │
    │  │ Step 5: 扩展课程学习指令范围                         │
    │  └─────────────────────────────────────────────────────┘
    │       每 1000 次迭代保存一次
                ▼
   logs/rsl_rl/unitree_Go2arm_flat/{时间戳}/model_XXXX.pt

③ 神经网络结构（actor_critic.py）

  输入（推理时）                    输入（训练时）
  hist_obs [N, 10×52]              priv_obs [N, 27]
       │                                 │
       ▼                                 ▼
  StateHistoryEncoder              priv_encoder
  ┌─────────────────────┐          ┌───────────┐
  │ Linear(52→30)       │          │ MLP       │
  │ Conv1d(30,20,k=4,s2)│          │ 27→32→18  │
  │ Conv1d(20,10,k=2,s1)│          └─────┬─────┘
  │ Flatten → Linear→18 │                │
  └─────────┬───────────┘                │
            │  latent [N, 18]            │ latent [N, 18]
            └──────────┬─────────────────┘
                       ▼
           prop_obs [N, 52] + latent [N, 18] = [N, 70]
                       │
                       ▼
                backbone MLP [256, 256]        ← 共享特征提取
                       │
          ┌────────────┴────────────┐
          ▼                         ▼
    腿部 Actor Head            手臂 Actor Head
    MLP [256, 128] → 12D       MLP [256, 128] → 6D
    （4腿×3关节）               （6个手臂关节）
                       │
                       ▼  合并
              actions [N, 18]

    Critic Head（价值估计，仅训练时使用）:
    backbone输出 → MLP [256, 128, 128] → 1D（状态价值 V）

④ 文件功能速查表

文件路径 角色 关键内容
scripts/rsl_rl/train.py 训练入口 启动仿真器 → 创建环境 → PPO训练循环 → 保存checkpoint
scripts/rsl_rl/play.py 推理入口 加载checkpoint → 重排历史obs → 推理 → 导出TorchScript(.pt)
scripts/rsl_rl/cli_args.py 参数工具 RSL-RL专用CLI参数定义（--checkpoint, --load_run等）
assets/go2arm_articulation_cfg.py 机器人硬件配置 USD路径、初始关节角、执行器类型/力矩限制/刚度阻尼
assets/go2_arm.usd 机器人3D模型 几何网格、碰撞体、惯量参数（PhysX使用）
tasks/.../go2arm_lab_env_cfg.py 核心MDP定义（基类） Scene/Event/Action/Obs/Reward/Curriculum 全部Manager结构
tasks/.../config/flat_env_cfg.py 平地训练配置 覆盖：平面地形、奖励权重调参、课程学习初始/最终范围
tasks/.../config/rough_env_cfg.py 复杂地形配置 覆盖：随机地形生成、启用push扰动、更大动作范围
tasks/.../config/agents/rsl_rl_ppo_cfg.py 算法超参数 网络层数、mixing_schedule[1.0,0,4000]、PPO clip/lr/epochs
tasks/.../config/__init__.py 任务注册 gym.register() 将任务名映射到cfg类路径
tasks/.../mdp/observations.py 观测函数 关节角/速度、本体加速度、接触力、特权obs（质量/力矩）
tasks/.../mdp/rewards.py 奖励函数 末端位置追踪(exp)、步态air-time、稳定性惩罚
tasks/.../mdp/pose_command.py 末端指令生成 在工作空间内随机采样7D末端位姿目标（含课程扩展）
tasks/.../mdp/velocity_command.py 速度指令生成 随机采样底盘速度目标（含课程扩展、静止比例）
env/manager_env.py 自定义环境类 继承IsaacLab ManagerBasedRLEnv，扩展obs分组逻辑
local_rsl_rl/modules/actor_critic.py 神经网络 双头Actor(腿/臂) + StateHistoryEncoder + priv_encoder
local_rsl_rl/modules/normalizer.py obs归一化 运行时均值/方差统计，推理时同步到policy.pt
local_rsl_rl/algorithms/ppo.py PPO算法 裁剪目标函数、价值损失、熵正则、梯度裁剪
local_rsl_rl/algorithms/distillation.py DAgger蒸馏 教师(priv_encoder)→学生(hist_encoder)的监督蒸馏更新
local_rsl_rl/runners/on_policy_runner.py 训练主循环 采集rollout → GAE → PPO更新 → 课程扩展 → 保存ckpt
local_rsl_rl/storage/rollout_storage.py 轨迹缓存 存储N_envs × N_steps的obs/action/reward/done数据
local_rsl_rl/wrappers/VecEnvWrapper.py 环境适配层 将obs字典拆分为(prop/priv/hist)三元组供网络使用
local_rsl_rl/networks/memory.py GRU记忆模块 可选的循环记忆网络（当前项目使用卷积历史编码器替代）
